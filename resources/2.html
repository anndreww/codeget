Ce limbaj sa folosec?

Cele mai populare limbaje sunt C/C++, Java și Python. C++ este un limbaj complex, folosit des. In schimb C este in esenta o versiune strict inferioară a C++ si nu are structurile de date incorporate care sunt adesea folosite.

In general, recomand urmatoarele: Daca stii deja una sau mai multe dintre aceste limbi, foloseste-o pe cea cu care te simti cel mai confortabil. Daca nu cunoasteti niciuna dintre aceste limbaje, ati putea la fel de bine să incepeti cu C++, deoarece utilizatorii de C++, in general, nu trebuie sa-si facă griji atat de mult ca solutiile lor sunt un factor constant prea lent. In plus, majoritatea modulelor nu au suport pentru Java si Python.
